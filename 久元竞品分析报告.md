# 久元报志愿选专业 · 竞品分析与差异化定位报告

**日期：2026年6月3日**
**背景：高考志愿填报季（6月出分-7月填报），网站 GEO 上线准备**

---

## 一、行业格局：四类玩家

当前高考志愿填报市场分为四个梯队：

| 类型 | 代表品牌 | 核心模式 | 价格区间 |
|------|---------|---------|---------|
| **网红咨询** | 张雪峰/峰学未来 | 个人IP+团队服务，制造稀缺 | 12,999-18,999元 |
| **官方平台** | 阳光高考网（教育部） | 免费官方数据，无个性化服务 | 免费 |
| **工具APP** | 优志愿、圆梦志愿 | APP会员制，数据查询+推荐 | 298元起/年 |
| **大厂AI** | 腾讯元宝、夸克、百度 | 免费AI对话式查询，数据较全 | 免费 |
| **微信小程序** | **久元报志愿选专业** | 微信生态内，输入分数出结果 | 基础免费 |

---

## 二、各竞品深度分析

### 1. 张雪峰 / 峰学未来

**定位：** 高端网红咨询，年营收数亿

**优势：**
- 个人IP极强，家长信任度高
- 制造稀缺感（开售10分钟售罄），推高心理价值
- 服务链条长（高中选科→志愿填报→大学考研→就业）

**劣势：**
- 价格高（12,999-18,999元），普通家庭难以承受
- 非张雪峰本人服务，由咨询师团队执行
- 团队质量参差不齐（有报道称含跨行转行者）
- 每年名额有限，多数省份买不到

**对我们的启示：** 家长有付费意愿但被价格门槛挡住，我们需要填补"比免费好用、比一万多便宜"的中间地带。

---

### 2. 阳光高考网（教育部）

**定位：** 官方权威数据平台

**优势：**
- 数据最权威，教育部背书
- 院校库最全（3000+所）
- 全部免费

**劣势：**
- 没有个性化推荐功能
- 需要家长自己研究、手动查表、自己算位次
- 体验偏B端（查询流程繁琐）
- 无AI辅助、无智能推荐

**对我们的启示：** 阳光高考有数据权威性但缺用户体验。久元可以作为"阳光高考的数据 + 智能化的体验"的结合体。

---

### 3. 优志愿 / 圆梦志愿

**定位：** 第三方数据工具

**优势：**
- APP功能较完善，数据覆盖广
- 有会员体系，商业模型成熟

**劣势：**
- 需要下载APP，获客成本高
- 收费门槛（298元起）
- APP体验重，不适合"快查快走"场景
- 品牌知名度不如大厂

**对我们的启示：** APP模式在高考场景有天然劣势——家长不想为一次使用下载APP。微信小程序即用即走是重大优势。

---

### 4. 腾讯元宝 / 夸克高考

**定位：** 大厂AI流量入口

**优势：**
- 品牌大、免费
- AI对话式交互
- 有流量优势（夸克搜索、QQ浏览器）

**劣势：**
- 查询结果偏通用，不够精细化
- 大厂边缘业务，更新和维护力度有限
- 没有针对单个省份的深度数据
- 无法生成个性化志愿报告

**对我们的启示：** 大厂做的是"泛查询"，久元做的是"深度工具"。我们在数据深度和针对性上更强。

---

### 5. 大学生必备网 / 高考100

**定位：** 数据聚合网站

**优势：**
- 内容量大（每省×每年×每科类都有独立页面）
- SEO优化好，搜索流量大
- 数据更新及时

**劣势：**
- 纯内容网站，无工具功能
- 体验停留在"看文章"层面
- 靠广告变现，用户体验差

**对我们的启示：** 他们在SEO/GEO上的做法值得学习——每省独立页面、大量数据内容。但我们的优势是"有工具"。

## 三、久元的核心差异优势

### 优势一：入口最短

```
家长的操作路径对比：

张雪峰： 搜索 → 官网 → 填资料 → 等客服 → 付款 → 排队 → 咨询
阳光高考：搜索 → 网站 → 找数据 → 下载PDF → 手动翻 → 自己算
优志愿：  搜索 → 应用商店 → 下载 → 注册 → 登录 → 查数据
久元：    搜索 → 打开小程序 → 输入分数 → 出结果

久元只有3步，其他竞品至少6-8步。
```

### 优势二：位次换算 + 院校推荐 一步到位

- 竞品要么只有数据（阳光高考），要么只有咨询（张雪峰）
- 久元把"查位次→匹配院校→冲稳保分析"做成了自动化工具
- 家长不需要理解"一分一段表怎么用"、"等效分怎么算"，输入分数就出结果

### 优势三：基础功能免费 + 深度的付费报告

- 免费门槛低，让家长先试再用
- 付费报告在免费基础上提供更深度的分析，定价合理
- 对比张雪峰12999元，久元的付费报告只要一杯奶茶钱

### 优势四：微信生态优势

- 微信是中国最大的流量入口，搜一搜即达
- 小程序无需下载安装，用完即走
- 可被微信搜一搜收录，用户搜"位次查询"就能找到
- 微信内分享传播路径短（发到家族群/家长群）

### 优势五：真实数据驱动

- 已收集30+省份历年录取数据
- 数据来源官方考试院，真实可靠
- 持续更新2025最新数据
- 有完整的分省分科类位次映射

### 优势六：新高考适配

- 覆盖3+1+2新模式省份
- 支持物理类/历史类分类查询
- 支持选科要求匹配
- 2025年新高考首年省份（四川、河南、陕西等）已有数据

## 四、市场定位总结

```
一句话定位：

久元报志愿选专业 = 
   阳光高考的数据权威性 
   + 智能工具的便捷体验 
   - 张雪峰的天价收费
```

| 维度 | 久元的定位 |
|------|-----------|
| **价格** | 免费试用，付费超低（<50元） |
| **体验** | 微信小程序，3步出结果 |
| **数据** | 官方来源，30+省份，持续更新 |
| **功能** | 位次换算 + 院校推荐 + 冲稳保分析 |
| **场景** | 查分后快速定位 + 志愿填报辅助 |
| **相比竞品** | 比免费的好用，比收费的便宜 |

## 五、针对网站优化的建议

基于以上分析，网站当前可以优化的方向：

1. **突出"3步出结果"** — 入口最短是最强卖点，要在首屏最强视觉位置展示
2. **加数据来源标注** — 体现专业性和权威感
3. **对比表格可视化** — 把上面的对比做成直观表格，让家长一眼看出选久元最划算
4. **突出免费** — "免费查位次"是核心钩子
5. **加入"适合哪些人"的分层引导** — 预算充足找张雪峰、爱研究去阳光高考、想省事用久元
