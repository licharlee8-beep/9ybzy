最后一天的检查清单

无论你是哪个省份的考生，最后一天请按以下顺序检查：

1. 确认提交状态（最重要）

登录系统，查看志愿状态。必须显示 "已提交" 而不是"已保存"。保存≠提交，每年都有考生因为只保存没提交而错过填报。

✅ 登录系统 → 查看志愿状态 → 确认是"已提交"

2. 检查保底志愿够不够

你的最后3个志愿，是否100%能上？如果不敢确认，说明保底不够深。保底志愿不要只填一所，要阶梯式保底。

各模式保底建议：

  • 专业+学校（河北96个/贵州96个）：最后10个志愿作为铁底
  • 院校专业组（江苏40个/吉林40个/云南40个/上海24个）：最后5个志愿作为铁底

3. 核对选科要求

系统不会自动拦截选科不符的志愿。检查每个志愿的专业选科要求是否匹配自己的选考科目。特别留意：

  • 理工科普遍要求"物理+化学"（2025年后明显增多）
  • 医学专业可能要求"化学+生物"或"物理+化学"
  • 部分专业接受"物理或化学或生物"三选一

4. 截图保存志愿表

提交后立即截图，发到自己的微信或邮箱做备份。如果之后需要修改，有截图做参考。

5. 保管好账号密码

不要在任何公共场所、班级群、朋友圈透露你的登录密码。近3年约12.7%的滑档案例与志愿信息泄露有关。

如果现在还没填好志愿

别慌，来得及。按这个流程操作：

  1. 微信搜「久元报志愿选专业」 → 选你的省份 → 输入选科和分数
  2. 系统自动换算位次 → 按冲稳保推荐完整的志愿列表
  3. 拿到推荐后 → 对照着填到官方系统里
  4. 提交 → 截图保存
今天是最后一天，但时间还够用：用工具生成推荐列表只需3分钟 → 填到系统里半小时 → 调整10分钟 → 提交。一个多小时就能搞定。不要因为觉得来不及就放弃，填了就有机会。

如果错过本科批怎么办？

即使错过本科批，还有以下出路：

  • 专科批：大部分省份的专科批7月下旬到8月初填报，专科同样有双高院校、专升本路径
  • 征集志愿：本科录取结束后，未录满的学校会发布征集志愿，这是最后一次上本科的机会
  • 复读：如果决心已定，了解复读的真实成本后再做决定

不管走哪条路，先把今天的志愿填了——填了才有选择权，不填什么都没有。
