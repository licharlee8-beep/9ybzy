# 主页最新文章更新提醒

## 问题

主页 `index.html` 的「最新文章」侧栏是**手动硬编码**的，不是自动从 `articles/` 目录读取的。每次新增/修改文章后，必须同步更新 `index.html` 的对应条目。

## 位置

`index.html` 约第 929~950 行，搜索 `最新文章` 即可找到。共 4 个条目，按日期倒序排列。

## 更新步骤

1. 写完新文章后，打开 `index.html`，找到「最新文章」区块
2. 把最旧的一条替换成最新文章（保持 4 条）
3. 检查日期倒序是否正确
4. **同时更新** `articles/index.html`、`sitemap.xml`、`llms.txt`
5. 确认没有重复条目（历史上出现过重复）
